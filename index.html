Envelope

The world of open source has matured and became an essential part of every developers life. But the world outside of software is missing out on this, and they shouldn't have to. Envelope aims to fix this.

Envelope is a place where people with coding skills can go to build a portfolio and share software tools with the rest of the world, while people without coding skills can go here and use these tools to create their own insights and their own visualisations, using their own input and their own data.

founder @ Envelope.li

Dashboards

Dashboard for Met het Net

Companies of today gather more data in a day than they used to do in years. This dashboard helps to quickly understand what is happening where. It shows data in tabular form, in an interactive graph, on a map, and it can all be downloaded in csv format.

This dashboard is developed for Met het Net, a Dutch startup in the energy sector.
